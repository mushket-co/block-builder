# BlockBuilder — React 18 + Vite Example

Тот же код, что в [`examples/react19`](../react19): `vite.config.js` указывает `root` на `../react19`, зависимости зафиксированы на **React 18**.

Используется для ручной проверки и E2E (`react18` Playwright project на `:3005`).

```bash
# из корня block-builder
npm run example:react18

# или
cd examples/react18 && npm run dev
```

Приложение: `http://localhost:3005`
